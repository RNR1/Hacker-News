# Hacker News Clone
This project is a clone of [Hacker News](https://news.ycombinator.com/) built with React & TypeScript.


## Installation

1. Clone or unzip the project and navigate to its folder.
2. Run `yarn install`.
3. Run `yarn start`.

## Tech-stack:

1. React
2. TypeScript
3. Styled Components
4. Axios
5. React Query
